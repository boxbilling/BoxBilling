Open Source billing software

Requirements
------------
You must clone the repository and then install dependencies using composer:

```bash
php composer.phar install
```

Requirements
================================================================================

* PHP >=5.3.3
* PHP extensions:
  * mcrypt
  * curl
  * zlib
  * PDO
  * gettext
* MySQL or any PDO compatible SQL server
*
